# Student-Management-System ( Spring Boot + MySQL )

Project Overview

Developed a backend-based Student Management System using Spring Boot to perform efficient management of student records through RESTful APIs. Implemented core CRUD operations along with pagination and sorting features using Spring Data JPA for optimized data retrieval. Integrated MySQL as the database for persistent storage and ensured proper request-response handling through RESTful web services. Verified all APIs using Postman to ensure correct functionality and reliability.

# Tech Stack:
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- RESTful Web Services
- Postman

# Features
- Create a new student record
- Retrieve all students
- Retrieve student by ID
- Update student details
- Delete student record
- Pagination support
- Sorting support

# Project Structure
- Controller: Handles REST API requests
- Service: Business logic layer
- Repository: Database interaction layer
- Entity: Database model

# API Testing
All APIs were tested using Postman to validate:
- Request handling
- Response correctness
- HTTP status codes
- Data accuracy
